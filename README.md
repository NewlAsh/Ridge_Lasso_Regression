# Ridge & Lasso Regression

This project demonstrates **Ridge (L2)** and **Lasso (L1)** Regression using Scikit-learn. It covers model training, regularization, coefficient analysis, feature selection with Lasso, and performance evaluation using metrics such as R², MAE, and RMSE.

The main concept was about dropping the columns which contribute almost nothing in the form of weight and may lead to contributing to overfitting.
